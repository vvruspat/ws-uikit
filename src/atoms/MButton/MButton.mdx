# MButton

A clickable element that triggers an action. Supports six visual modes, three sizes, icon slots before/after the label, and a disabled state.

## Modes

| Mode | When to use |
|------|-------------|
| `primary` | Main call-to-action — highest emphasis |
| `secondary` | Secondary action |
| `tertiary` | Low-emphasis action |
| `outlined` | Outlined style with visible border |
| `transparent` | Ghost button, no background |
| `round` | Icon-only circular button (FAB) |

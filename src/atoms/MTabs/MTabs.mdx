# MTabs

A tab navigation component for switching between different views or content sections. Tabs can include icon slots and support a disabled state per tab item.

---

## Overview

<Canvas of={TabsStories.Basic} />

---

## Features

- **Icon slots** — Add icons before tab labels via the `before` prop on each tab item
- **Disabled tabs** — Individual tabs can be disabled by setting `disabled: true` on the tab item
- **Content area** — Each tab displays its associated `content` when active
- **Default active** — Control the initially active tab via `defaultActiveKey`
